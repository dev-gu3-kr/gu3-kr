# cat-form-rules

## 적용 범위
- 공지사항에 한정하지 않고, **모든 폼 UI**(작성/수정/설정/필터)에 공통 적용한다.

## 1) 폼 구현 원칙
- 폼 상태 관리는 React Hook Form(`react-hook-form`)을 기본으로 사용한다.
- `useForm`의 `mode`는 기본적으로 `onSubmit`을 사용한다.
- 서버 스키마 검증과 별개로, 클라이언트에서 즉시 피드백 가능한 기본 검증을 함께 구현한다.

## 2) 필수 항목/검증 규칙
- 필수 필드는 `trim()` 기준으로 공백 입력을 허용하지 않는다.
- 선택 필드는 명확히 optional로 유지한다.
- 필드별 검증 메시지는 짧고 사용자 친화적으로 작성한다.

## 3) 오류 UI 규칙
- 필수 라벨에는 빨간 `*`를 표시한다.
- 검증 실패 필드는 빨간 outline/border로 표시한다.
- 입력값은 검증 실패 시 유지되어야 한다(실패했다고 필드가 초기화되면 안 됨).

## 4) 에디터/커스텀 입력 동기화
- Toast UI 같은 커스텀 입력은 RHF 필드와 즉시 동기화한다.
- 동기화 시 `setValue(name, value, { shouldValidate: true })`를 기본으로 사용한다.
- 라우트 전환 시 cleanup으로 런타임 에러가 나지 않도록 안전하게 정리한다.

## 5) 제출 성공 UX
- 성공 시 `sonner`(`toast.success`)로 명확한 피드백을 제공한다.
- 성공 토스트 후 이동이 필요한 경우, 대상 페이지로 라우팅한다.
- 실패 시에는 에러 메시지를 유지하고 입력값은 보존한다.

## 6) API 메서드 일치
- 클라이언트 fetch builder 메서드와 route handler HTTP 메서드를 반드시 일치시킨다.
  - 예: `POST` 작성, `PATCH` 수정, `DELETE` 삭제
- 메서드 불일치 시 기능은 정상처럼 보여도 저장/수정이 실패할 수 있으므로 우선 점검한다.

## 7) 품질 게이트
- `src/**` 변경 후 `pnpm run validate`를 반드시 통과시킨다.
- 경고/오류 0 원칙을 유지한다.
