# cat-systematic-debugging

## 절차
1. 재현 조건을 한 문장으로 고정
2. 경계 분리(app router / feature server / client)
3. 가설 1개 수립
4. 관측(로그/assertion) 추가
5. 최소 수정
6. 회귀 방지(테스트/문서)
7. 검증: `pnpm run validate`

## 출력
- 재현 스텝
- 원인(근거)
- 수정 파일
- 회귀 방지 방법
